1. Switch Doodlenauts.json ABI to your own contract's ABI

2. Switch the address to your contract's addres in App.js

3. Go to blockchainActions.js and switch networkId to 1, and put in your smart contract address on line 54.

4. Change package.json homepage to https://boocrew.io/mint.

5. Build! type "yarn build" in cmd prompt that is in directory of boocrewminter. This will create a "build" file in the boocrewminter.

6. Put "build" file from boocrewminter into static website repo and rename it to "mint" instead of "build".

YOURE ALL DONE! let me know if you have any questions.
